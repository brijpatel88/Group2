# AI Reflection — Brijesh Patel

## 1. How did you use AI during this project?

I used AI as a learning and support tool throughout the project. Since I am still improving my React and frontend development skills, AI helped explain concepts, troubleshoot issues, and provide guidance when I became stuck. It was especially helpful for understanding component structure, UI improvements, responsive styling, and debugging problems.

AI also helped simplify technical topics into smaller and easier-to-understand steps. In some situations, it helped me better understand Git and GitHub workflows, such as working with branches, merging updates, and syncing changes in a team environment.

Another helpful part was using AI to review ideas and improve the overall user experience of the application. It helped identify ways to make the interface cleaner, more modern, and more responsive.

## 2. How did you verify or adapt AI-generated code?

I reviewed and tested AI suggestions before applying them in the project. Instead of using suggestions exactly as they were, I adjusted changes to make sure they matched our project structure, styling, and assignment requirements. Since this was a team project, it was important to make sure changes worked correctly with the existing codebase and did not affect other features.

I also tested features after making changes to confirm everything still worked properly. For example, after UI updates or styling changes, I checked whether functionality such as filters, task management, responsiveness, and navigation still worked as expected.

This process helped me better understand the logic behind the changes and increased my confidence in solving issues independently.

## 3. What did you learn about using AI effectively?

One important thing I learned is that AI works best as a support tool rather than a replacement for learning. Asking clear questions and understanding the responses helped me get better results. I also learned that not every suggestion fits perfectly into a project, so reviewing and testing changes is very important.

Another thing I learned was that AI can speed up problem-solving and improve productivity, especially when learning new technologies. However, personal understanding and decision-making are still necessary because developers must understand how and why something works.

Overall, this project helped me see AI as a useful learning assistant that can support development, improve understanding, and help solve problems more efficiently when used correctly.
